# Gulp Boilerplate for HTML/CSS/JS development
## Structure
* public
    * src (developer working folders)
        * css
            * scss
                * common
                * mixins
                * pages
        * img
            * sprites
                * etc (sample)
                * icons (sample)
                * txt (sample)
        * js
            * vendors
        * views
    * dist (desination folders. it auto building when you'are first run gulp)
## How to use
after install nodejs https://nodejs.org/ko/
> 1. npm install
> 2. gulp
