# Guide gulpfile.js

## 전체 동작 프로세스
    1. gulp 구동 시 리소스(resources) 초기화 진행
    2. server 를 띄우고 초기 설정해둔 파일 형식 또는 경로를 주시(watch)함
    3. HTML / JS / IMAGE / SCSS / LESS 파일의 변경이 있을 시 각각의 task(작업)를 실행함.
